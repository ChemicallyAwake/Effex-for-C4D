MOP::Cacher

Parameters Constraints
MOP::Cacher

The mesh cacher can save/load any Implicit Mesher to/from disk.

Usually this is not necessary (and you probably even don't want to cache the mesher if you are not using it for foam, spray & bubbles) because the implicit mesher can operate on the fly on cached simulation data.

This even works in network rendering because the mesher doesn't require any information from previous frames but only the current data. By not caching the mesher you save valueable simulation time.
