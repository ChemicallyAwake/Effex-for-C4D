Mesh

The mesh node converts a Cinema 4D polygon mesh to an Effex triangle mesh so it can be used by other nodes in the Effex framework.
UV data and custom normals (normals tag) from the C4D mesh will automatically be transfered.

An Effex mesh has the following properties:
1. It is a triangle mesh (no quads or n-gons exist)
2. It can hold arbitrary amounts of vertex properties

Mesh vertex properties can be accessed with the same Property nodes that are used to access particle properties.
