CS::Distance To Object

Parameters
CS::Distance To Object

The Distance to Object constraint uses the distance to the voxel/grid cell from the constraint position itself or from a given object.

It's great to let things start happening when a certain object reaches that area (for example when a character enters a hallway and once he passes torches, they start to burn...easily done with this constraint).

It's also great for use in cell operators.
