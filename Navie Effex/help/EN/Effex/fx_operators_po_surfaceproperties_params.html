Particle Group





The particle group to calculate the properties for.

Particles





The particle type of the Particle Group to use.

Fluid Dynamics





Currently required in order to calculate normal and curvature of particles. Link any fluid dynamics node here.

Surface normal





Calculates and attaches a surface normal property.

Curvature





Calculates and attaches a curvature property.

Neighbor Anisotropy





Calculates and attaches a particle anisotropy property. If you don't use a thin sheet that calculates anisotropy, you can use this one in order to use it in the anisotropic mesher (or for any other purpose).

Sampling Radius





To find particles nearby this radius defines the maximum distance around each particle to find neighboring particles.

Min. Density





Defines the minimum density a particle must have to be included.

Max. Density





Defines the maximum density a particle must have to be included.
