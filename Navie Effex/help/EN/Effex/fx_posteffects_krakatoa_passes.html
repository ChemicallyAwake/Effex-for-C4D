Basic Precomputations Rendering Passes
Passes

Krakatoa can output several other passes and Effex can write them into a Cinema 4D multipass for compositing (which is much more powerful than direct rendering).

In order to let them be created you need to add the 'Post Effects' multipass.
Effex will then create new passes and write the Krakatoa passes into them.

Normal



Outputs a pass containing the normals of the rendered particles.

Velocity



Outputs a pass containing the motion vectors of the rendered particles.

Z-Depth



Outputs a pass containing the depth of the rendered particles.

Occluded RGBA



Outputs a pass containing the occluded particles.

The Occluded Particles pass includes all particles that are partially or completely occluded by matte objects, while the main Beauty RGBA pass will contain all the particles completely in front of matte objects.
