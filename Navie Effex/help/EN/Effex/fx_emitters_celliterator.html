Emitter::Cell Iterator

Parameters
Emitter::Cell Iterator

The cell iterator gives you access to Grid Channels on a cell level.
It iterates each grid cell of a Grid Channel and evaluates COPs (Channel OPerators) for each. This allows very complex operations.

Note:
If you wonder "what's the difference to the Emitter::Grid then?":

The Emitter::Grid is also running over all grid cells BUT you can only specify a single value (in the emission settings) that is emitted into all cells. The Cell Iterator simply allows deeper and local access instead of the Emitter::Grid's global approach.
