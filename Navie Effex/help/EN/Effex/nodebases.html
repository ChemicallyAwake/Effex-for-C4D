Node Display (Live Inspector) Constraints
Bases

Base tabs are used globally by many or all nodes.
These base tabs will always contain (at least) the same settings, so they are only documented once here in the documentation.

For example all Effex nodes (= Effex objects in the OM) that are directly executed (= shown in the Pipeline Viewer) by the framework have a 'Node' tab.
It contains general information about an Effex node and its execution behavior when the framework processes the pipeline.

If a certain node has additional settings in one of the base tabs it will have an according base tab page containing the additional settings only.
