RAM Cache

The RAM cache will store the rigid-body matrix all the time (no matter if you use RAM cache or not). So each time you ran a simulation it is stored automatically.

Duration





Takes a node of type 'duration' in order to define how long the RAM caching should take place.

Use RAM cache





Activate using RAM cache.

Save subframes





Subframe positions will be stored as well if checked.

Clear Cache




Clears any available cache.

RAM Cache::Load

Use Cache





If there is a cache, you can load it by activating this option.

Offset [Frames]





Offsets the loaded cache frame.
