GOP::Dissipation

Parameters
GOP::Dissipation

Allows to dissipates any values of a Grid Channel (smoke for example).

Note:
You could achieve the same with an Emitter::Cell Iterator and a COP::Math in substract mode but as this is used a lot in fluid simulation work it is more straightforward to have a dedicated node.
