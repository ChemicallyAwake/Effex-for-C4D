CS::Grid Value

Parameters
CS::Grid Value

The Grid Value constraint reads out a value from any Grid Channel and remaps the value which is then used as intensity for the calling node operation.

It can be used to allow operations of other nodes only if a certain type of data is available. For example to emit smoke according to the temperature (although the combustion node does this automatically).
