Parameters

Particle Group





The particle group to calculate the properties for.

Particles





The particle type of the Particle Group to use.

Fluid Dynamics (Boundary)





Link a fluid dynamics node in order to get the domain boundary being taken into account for any simulation particles.

Duration





Takes a duration node in order to control when of the weak spring operation is evaluated.

Iterations



Define how many times the operator should be executed.
If you use more than 1 iterations, you may want to decrease the stiffness parameter to get a softer displacement of the particle positions.

Particle Spacing



Defines the particle spacing of the simulation. The default is 0.5 (half a voxel size).

Note: Usually you won't change this except you know exactly what you are doing!


Spring stiffness





The spring stiffness. Be careful with this. The lower the time step (or rather the more substeps) the higher you can set the stiffness without it getting unstable.

Enforce Resampling





Especially at boundaries particles can clump together. This option tries to resample the particle positions to avoid numerical issues.

Note: It should used with care and probably only in highly dynamic scenes as it may show popping.


Mimic Boundary Particles





During displacement of positions in this node, collisions with boundaries are included (if a Fluid Dynamics node is assigned).
Activating this option will basically try to offset the particles from the collision surface, however it does this with an impulse rather than displacement.

Can be used arbitrarily if you see fit but is also most useful for dynamic scenes or if you need improved slipping behavior at boundary edges.
