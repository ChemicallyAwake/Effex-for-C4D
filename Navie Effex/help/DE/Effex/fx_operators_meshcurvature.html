MOP::Curvature

Parameters
MOP::Curvature

The MOP::Curvature node calculates mesh surface curvatures which are stored as mesh vertex properties.

These properties are the same as particle properties. You can use the same Property::Curvature node to access the mesh vertex curvature after MOP::Curvature has been evaluated.

All you need to do is add a MOP::Curvature to an Effex Scene and link the mesh node to calculate curvatures for.
Even more conveniently: each mesh node has a Quicktab button to do exactly that.

This curvature value is for example used by the Emitter::Particle On Mesh to allow curvature based emissions.
