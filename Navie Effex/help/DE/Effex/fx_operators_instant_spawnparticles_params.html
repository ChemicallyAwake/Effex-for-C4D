Documentation : Effex : Nodes : Operators : Particle Instant Operators (PIOP) : PIOP::Spawn Particles

Parameters

Parameters
Parameters

Particle Target Group





The particle group to emit the spawned particles to.

Particles





The particle type of the Particle Group to use.

Duration





Takes a duration node that defines when the spawning instant is allowed to operate.

Seed





The random seed for any randomize values in this node. A seed value is like a start location inside of a random sequence.

Count





The amount of spawned particles.

Count (Rendering)





The amount of spawned particles in rendering (to picture viewer).

Randomize





Randomize the particle count each time a new spawning cluster is generated.

Lifetime





Defines how long spawned particles survive.

Randomize





Randomize the life time of spawned particles.

Angle





Defines the spawning angle along the spawning normal (if the particle for example spawn on a surface, it's the surface normal).

Randomize





Randomize the spawning angle of spawned particles.

Velocity of original





Defines a multiplier for the initial velocity (taken over from the original particle that triggered the spawning) of the spawned particles.

Randomize





Randomize the initial velocity of spawned particles.
