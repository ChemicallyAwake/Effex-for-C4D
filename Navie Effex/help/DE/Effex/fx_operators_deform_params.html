Particle Groups



Define the particle group containing the particles to deform.

Particle Property



By default POP::Deform will store the deformed position virtually in a particle property named "Deformed Position".
If you want another property to take the deformed position you can drop it here.

Using a virtual position instead of directly modifying the real particle position makes the whole operation non-destructive.

Duration



Define when and how long this node operates.

Time integrated



Add the deformations time integrated (so the deformation is per second otherwise per frame).

Particle type



Define the particle type (subgroup) of the linked particle group to deform.

Math Operation



Define how the deformation is applied to the property.

1. Set - Overwrite
2. Add - Add

Intensity



Global blend or mix of the original position with the deformed position.
