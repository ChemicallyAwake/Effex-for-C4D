Parameters



These parameters are for convenience only as the Particle Group itself is only a particle container. But for quick rendering of the according particles in this group you will find some settings here.

If you want more control over rendering particles you should use the ParticleGeometry object to render sprites or any kind of geometry attached to particles.

Furthermore, if you have a Thinkbox Krakatoa license, you can use our Krakatoa renderer bridge.
In the future, Effex will get its own particle rendering solution.
