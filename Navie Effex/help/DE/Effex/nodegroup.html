Node Group

The Node Group object does only contain a few settings.
The purpose of the Node Group is as the name already suggests to group Effex nodes for your own or user convenience.

It will display all settings of grouped nodes in its own interface allowing you to create so called High-Level nodes (comparable to Houdini assets but you still have full access to the low-level without switching applications!)

How to use

Drop any node you would like to group under the Node Group object in the Object Manager.
The dropped node will vanish from the object manager.

Note: You can stop a node from being identified as a node to group if you attach a 'Stop' tag to it!

The dropped node's attributes will now be added to the Node Group attributes.

Note: You can then style the Node Group's interface with the Node Styler dialog.
Double click on the Node Group's icon to open the Node Styler.

The order in which the nodes are in the child hierarchy will also be the order of node subgroups in the Node Group interface.

Note: As the interface is dynamically updated you should first make sure the order of nodes is indeed the one you want. Changes to the child hierarchy order will reset any changes made in the Node Styler!
