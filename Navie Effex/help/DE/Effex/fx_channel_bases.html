All channels provide the user with a 3d grid. It's size is usually determined by the currently available simulation data. For example it is extensively used in smoke simulations but also liquids use them and in the end it's just a grid that you can fill with any data you want to!



The grid is made up of grid cells. Each grid cell stores a scalar (floating point) or vector (3 scalars) value.
You can fill and modify grid data with the Grid Emitters (or via the Effex Xpresso nodes in Cinema 4D) and/or any existing grid operators (GOPs).
