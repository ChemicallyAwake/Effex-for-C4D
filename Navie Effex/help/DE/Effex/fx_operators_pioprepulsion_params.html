Parameters

Duration





Takes a node of type 'duration' in order to define how long this node should operate. See Duration.

If no duration is linked, it will operate indefinetly.

Use Position Velocity





If you are using a position based particle simulation you must activate this option.
In order to correctly calculate the repulsion force as a position update.

Bounce





The amount of bouncing produced.

Slide Drag





The amount of drag experienced by the collided particle when it slides along the surface.
Negative values are only available for artistic purposes!
