Parameters

Particle group



Define the particle group with the particles to modify.

Target Property



Define the target property to modify. If no such property is attached to the particles nothing will happen.

Source Property



Define the source property to read and then write into the target property. If no such property is attached to the particles nothing will happen.

Duration



Define when this node operates.

Data type



(Only available if no 'Source Property' is defined)

Define the data type of the 'Value' to modify the target particle property with.
This value can either be a scalar (floating point), a vector or an integer value depending on 'Data type' setting.

Value



(Only available if no 'Source Property' is defined)

Define a value to change the target property with (according to 'Math. Opertion' setting).

Particle type



Define the particle type (subgroup) of the particle group to access the property from.

Math. Operation



Define how the source property or Value is applied to the target property using the selected math operation.

Note: The 'Dot Product' operation makes only sense for vector properties and will result in a scalar value.
