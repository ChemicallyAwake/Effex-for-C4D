Display

Quicktab Parameters Display
Display



All settings here are only applicable to the editor views and hardware renderings.

On



Turns on/off the display for this particle group and all its subgroups.

Display::Passive Particles
Contains all the settings for the display of passive particles in the editor view.

Draw each Xth particle





Allows to set an interval for drawing particles. A value of 0 will turn off the display of particles.
A value of 1 will display all particles, a value of 2 will draw each second particle and so on.

Display Size



The display size in the editor view or during hardware rendering.

Color





The display color of this particle type.

Note: This does NOT change the Color property of the particles but is a separate color only used for display (not for simulation).


Deformed Position



Draws the deformed position of the particles instead of the real position.

Draw Radius



Draws the particle's radius as a circle/sphere. But only if the particles have a Radius property attached!

Draw Radius in 2D



Draws the particle radius as a camera facing circle instead of as a sphere.
It is faster than drawing as sphere but slower than drawing as a point.

Property



Here you can assign (either manually or via its candidate button) a particle property that should be used for coloring the particles.
If the particle has no such property attached (you can attach properties directly in the according particle emitters) then the particles are displayed in black (no color).

Threshold [voxel size]





Here you can set a threshold value (the value is given in physical space) to quickly range map the property value (while the threshold is the maximum value allowed).

Example:
Imagine the velocity property is linked. A velocity vector value could be arbitrary in length, depending on the scene.
Let's say you have maximum velocity lengths (speed) of 40m/s, then a threshold of 40 is the best option (so 40/40 = 100%). In that case the particles would be completely white (full intensity).
