Display

Quicktab Parameters Characteristica Dynamics Solver Display
Display

On



Turns On/Off the drawing of this node.

Show Adaptive Bounds





Displays the adaptive grid bounds in the editor.

Info





Shows some general information about the resulting grid fitting into the Dimensions setting (so these are max. values) given the current voxel and container size.

Cells: the amount of cells along each direction followed by the overall cell count.

Cells Rendering: the amount of cells for rendering along each direction followed by the overall cell count.

Container size: the Dimension setting given in the Effex framework units (see Effex Scene)

Cell size: Size of a voxel in C4D space units.

Velocity drawing plane





Turn ON/Off the drawing of the velocity field motions in the editor.

You can define along which axis the drawing slice is displayed.
Use the "Relative Position" to define where the slice is drawn along that axis.

Relative Position





The relative position of the drawing slice inside the (adaptive!) dynamics container bounds.
50% means in the center along the currently selected slice direction.

Dot display





If checked the velocities are only displayed by dots instead of lines.

Minimum velocity [cells]





The minimum velocity magnitude shown. The unit is given in voxels. So a value of 1 means the length of 1 voxel size.

Maximum velocity [cells]





The maximum velocity magnitude shown. The unit is given in voxels. So a value of 1 means the length of 1 voxel size.

Color





The display color of the velocity field.

Velocity driven Transparency





Activate to let the editor display transparency be controlled by the velocity magnitude itself while high velocities mean less transparent. Zero velocity means fully transparent.

Obstacles Cells





Turn ON/Off the drawing of the cells that are defined as obstacles in the editor.

Grid





Turn ON/Off the drawing of the grid plane in the editor.
