Parameters

Spline



Drop here the spline based object that acts as a path for particles to approach and follow.

Particle groups



Drop here any particle groups to include.

Target property



This is optional. If you want another property than the position to be updated drop it in here.
It will contain the new position of the particle.

Duration



Define when and how long this node operates.

Start Position



Defines the location on the path where particle should start following it. It is determined in the first frame where this particle is seen by the operator (see 'Distance').

1. Nearest Point - starts at the nearest spline point to the particle
2. Nearest Vertex - starts at the nearest spline vertex to the particle
3. Vertex Index - user can specify a specific spline vertex index to start from
4. Percentage - user can specify a location on the spline with a percentage value (along length of spline)
5. Random - starts at a random position on the spline

Vertex Index



(Only available if 'Start Position' is set to 'Vertex Index')

The vertex index to start from.

Percent



(Only available if 'Start Position' is set to 'Percentage')

The position along the spline to start from given as a percentage value. 0% means at the first vertex, 50% means exactly at the middle of the path and 100% at the end (or at the first point in case the spline is closed).

Segment



(Only available if 'Start Position' is set to 'Vertex Index' or 'Percentage')

Define the spline segment to use.

Segment Index



(Only available if 'Start Position' is set to 'Vertex Index' or 'Percentage')

The index of the spline segment to use. See 'Segment'.

Distance



The maximum distance for particles to check for a spline path point to start from. If further away they are not moving to the spline.

Parameters::Move To Spline
Settings to define how particles move to their starting position on the spline path.

Original velocity



Defines that the maximum velocity for a particle to move to the spline may not exceed the particle's original velocity when it was detected in range of the spline.

Max. Velocity



(Only available if 'Original Velocity' is unchecked)

Defines that the maximum velocity for a particle to move to the spline may not exceed this value when it is detected in range of the spline.

Acuteness



The acuteness or urge to move to the spline. Higher values make the particles move to the starting position more directly.
This value can be set higher than 100% if required.

Intensity



The overall intensity of the movement to the spline.

Parameters::Follow Spline
Defines the path following behavior once particles reached their starting position.

Blend position



Blend the follow spline position with any original particle position. Original means that you could have other forces etc. acting on the particles which would have no effect if this is set to 100% (fully following).

Randomize



Randomizes the blend position value.

Offset



This is the offset from the spline along the spline path's normal.

Randomize



Randomizes the offset from the spline.

Rotation



Allows to rotate the normal (used for the Offset) along the spline path.

Randomize



Randomizes the Rotation of the path normals along the spline.

Step rotation



Rotates the particle on each (sub)step around the spline path by the given angle.

Randomize



Randomizes the Step Rotation.

Slope



You can change the slope of the particle direction by the given angle.

Randomize



Randomizes the Slope.

Loop mode



Defines what happens once a particle has reached the end of the spline.

1. Restart - restarts moving to its starting position on the spline
2. Loop - keep on following from the beginning of the spline
3. Finish - finish the following and detach from being processed

Direction



Defines if the following happens in forward or backward direction along the spline.

Velocity



Define the particle velocity while following the spline path.

1. Entering Velocity - Use the velocity of the particle when it reached the starting position
2. Specific Velocity - Use a custom velocity
3. Frames for circumnavigation - Use a certain amount of frames for the particle to circumnavigate the spline path.

Intensity



An intensity multiplier for the chosen 'Velocity' type.

Custom



(Only available if 'Velocity' is set to 'Specific Velocity')

Defines the custom speed.

Frames



(Only available if 'Velocity' is set to 'Frames for circumnavigation')

Defines the amount of frames required for circumnavigation. The particle velocity is calculated automatically from this.
