Constraints

Constraints





Restrict which particles are included in the reconstruction by using constraints.

Please see Constraints for more information.

A very useful choice here is the Particle count constraint, to let particles with only very few neighbors not be included.

Use Particle Density





Activates the exclusion of particles based on their particle density (to only use particles near the surface and not inside the liquid body where no surface is reconstructed anyway).
This can speed up the process especially in higher resolutions with a lot of particles.

Density Threshold





Is only used if 'Use Particle Density' is activated and gives the density value limit before particles are excluded from the reconstruction.
So any particle that has a density property value higher than this threshold will be excluded.

Max. Distance





Is only used if a 'Surface Tracker' (liquid/viscous dynamics) is linked and allows to exclude particles based on volume depth, excluding particles inside the liquid body and only use the ones near the surface. This can speed up the process especially in higher resolutions with a lot of particles.
