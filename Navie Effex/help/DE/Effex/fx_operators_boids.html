POP::Boids

Parameters
POP::Boids

The boids algorithm is a well-known, well-working bird flocking algorithm.
So it tries to make sure that particles stay in flocks or clusters (like birds do).

Furthermore the operator allows to define predator and prey particles. Prey particles will always try to escape from predators so they are not eaten.

In the end you can do nice motion graphics stuff with this operator.
