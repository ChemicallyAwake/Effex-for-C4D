Outports

Cell Index


Contains the converted position in Cell Index space. The Cell Index is always between 0 and Cells Count.

Note: Requires 'Reference space Channel' to be linked in order to correctly get the cells count.

Cell Coordinates


Contains the converted position in Cell coordinates space.
Cell Coordinates are a vector between (0,0,0) and (cells count x, cells count y, cells count z).

Note: Requires 'Reference space Channel' to be linked to get the amount of cells along each axis.

Global C4D space


Contains the converted position in Global C4D space.

Global Physical space


Contains the converted position in Global Physical space (given by the linked Effex Scene or the linked Grid Channel).

Local C4D space


Contains the converted position in Local C4D space.

Note: Requires 'Reference space Channel' to be linked in order to correctly convert into the channel's local coordinates.

Local Physical space


Contains the converted position in Local physical space.

Note: Requires 'Reference space Channel' to be linked in order to correctly convert into the channel's local coordinates.
