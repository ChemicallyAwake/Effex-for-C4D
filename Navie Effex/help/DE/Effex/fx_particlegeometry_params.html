Parameters

Particle group



Defines the Particle Group to read particles from.

Duration



Define when (temporally) and how long this node operates.

Only during rendering



If activated objects/geometry are/is only generated during rendering.

Particle type



Define which particle type (subgroup) of the input particle groups to use.

Position Type



Define which particle position to use.

Position
The real particle position.

Deformed Position
The deformed particle position (i.e. deformed by a POP::Deform)

Weighted Position
The weighted position (i.e. calculated by a POP::Volume Properties)

Clones



The input (child) objects are cloned for being attached to particles. This defines the type of clone generated.

Please refer to the Cinema 4D help for more info on Instances.


Alignment



Defines the alignment of attached clones on the particle.

X/Y/Z Axis
Clones are aligned along one of the global axis directions.

Travel direction
Clones are aligned along the travel direction of the particle they are attached to.

Look at Camera
Clones are always facing the camera. This mode can be used for sprites rendering.

Parameters::Scale

Uniform Randomizing



If activated randomizing the scale happens with the same value for each scale axis (X/Y/Z).
Otherwise you can define scale for each axis separately.

Scale



The scale of a clone along each axis.

Randomize X



Randomizes the 'Scale' along the X-axis.

If 'Uniform Randomizing' is activated, it also controls the scale along the Y and Z axis.

Randomize Y



Randomizes the 'Scale' along the Y axis.

Randomize Z



Randomizes the 'Scale' along the Z axis.

Motion Scale



Controls the scaling along the particle's travel direction.

Randomize



Randomizes the motion scaling factor.

Parameters::Rotation

Rotation



Controls the rotation of clones along each axis.

Randomize



Randomizes the clone rotation.
