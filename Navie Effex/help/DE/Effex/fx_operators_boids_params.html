Parameters::Prey
Prey particles always try to stay together in flocks (cohesion) but also try to avoid each other (separation).
Furthermore they try to fly in the same direction (alignment) and they try to escape any existing predators in their vicinity (fleeing).

Prey Particle type



Define which particle type (subgroup) of the prey particle group to use.

Max. Velocity



The maximum velocity a prey particle can reach.

Cohesion



Urge of prey particles to stay close to each other.

Cohesion distance



Each prey particle tries to stay close to other prey particles in its vicinity. This is the maximum distance to check for other prey particles.

Separation



Prey particles try to avoid collisions with other prey particles, so they need to separate. This defines the urge to separate.

Separation distance



The distance to check for close particles to separate from.

Alignment



Prey particles try to fly in the same direction as the rest of the flock. This defines the urge to align the flight direction.

Alignment distance



The maximum distance to check for other prey particles in order to align to their flight direction.

Fleeing



(Only available if Predator particle group is defnied)

Prey particles try to flee from Predator particles. This defines the urge to flee.

Fleeing distance



The maximum distance to check for predators in order to escape their (possibly) deadly attack! :)

Parameters::Predator

Predator Particle type



Define which particle type (subgroup) of the predator particle group to use as predators.

This must not be the same as the Prey::particle type if it is the same particle group

Max. Velocity



The maximum velocity a predator particle can reach.

Prey chasing



The urge to chase prey particles in the vicinity.

Prey chasing radius



The maximum distance to check for prey in order to chase it.

Predator separation



Predator particles try to avoid collisions with other predator particles, so they need to separate. This defines the urge to separate.

Separation distance



The distance to check for close particles to separate from.

Eat prey



Kill prey particles once the predator reaches them.

Distance



The distance to a prey particle to consider it caught (and ready to be eaten = killed).
