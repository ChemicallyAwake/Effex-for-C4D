Constraints





Restrict and alter the strength of the resulting emission by using constraints.

Please see Constraints for more information.

Apply Constraints to..





Particles
The defined constraints are only evaluated for the particle positions. Means that only particle constraints (PCS::) of the list are evaluated.

Particles & Grid
Same as 'Particles' but all(!) defined constraints are evaluated for the target grid location.
This allows to use grid constraints (CS::) and particle constraints (PCS::) to control the transfer result at the grid location.
