MOP::Smoother

Parameters
MOP::Smoother

This node calculates area-weighted smoothing on any Effex generated mesh. You will probably mostly use it to smooth Implicit Meshers.

Make sure that your MOP::Smoother is executed AFTER the implicit mesher. Use the Pipeline Viewer to make sure this is the case.
