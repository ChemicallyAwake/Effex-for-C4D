Particle Groups Mode



Define if the "Particle Groups" list will include the specified groups or exclude them.

Particle Groups



Drop any particle groups the forces should be applied to in here.

Forces



Drop any particle forces (PF or DF) in here that should be applied to the given particle groups.

Particle Property



By default the Particle Property::Velocity will be altered with the applied forces. But here you can define a custom vector property to take the forces.

Duration



Define when the forces should be applied with a duration node.

Time integrated



The forces are applied time integrated.
In other words: if checked the forces are applied per second, otherwise per frame.

Math Operation



Define how the forces are applied (how the target velocity or particle property is changed).

Add: The forces are added to the particle's existing velocity (property) value
Set: The forces are overwriting the particle's existing velocity (property) value
