Particle Group

The particle group is a container for any Effex particles that are generated inside of the framework (so by any other node).

Overall there are four different predefined particle types (and therefore four subgroups in the particle group) with different purposes but they are not restricted to these purposes. It's only for user convenience that these particle types are defined but you can use them, like any other part of the framework, for any purpose you want.

PASSIVE PARTICLES
Passive particles to create ballistic particle systems.

LIQUID PARTICLES
Liquid Particles are specifically used by any liquid simulation (FLIP, SPH...).

VORTICITY PARTICLES
Vorticity particles are explicitly used by the GF::Particle Turbulence node.

DIFFUSE PARTICLES
Diffuse particles are used to simulate secondary particles like foam, spray and bubbles (from a large-scale point of view so these are not meant to simulate close-up versions).

The POP::Foam/Spray/Bubbles node will specifically create and modify diffuse particles.
